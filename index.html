Programm Work Wize Website

Workwize is a fully integrated, project that represents the skills I've learned in the software engineering Bootcamp. It's a project and task management system designed for businesses and teams to streamline workflows efficiently. The application allows users to create workspaces, manage tasks, assign team members, and monitor progress through an interactive dashboard. The backend was developed using Python, the Django REST framework (DRF), and PostgreSQL, while the frontend was built using React and Tailwind CSS, with JWT authentication for secure access.

Design Riyadh First Health Center Website

This web application is designed to help healthcare centers efficiently manage and monitor their workforce, especially nursing staff. The platform provides real-time insights into each center by tracking the total number of employees, performance metrics, absences, workload distribution, and top-performing staff. It also offers a detailed dashboard with analytics such as frequent absences, center evaluations, nursing shortages, top paths, and recent activities. The system helps administrators make informed decisions, improve workflow efficiency, and maintain high standards of care across all centers. "

Designing Tadark App

Tadark is Smart Medication Error System that helps hospitals document and track medication errors easily using NFC and AI. It allows pharmacists to report errors quickly, sends instant alerts to doctors, shows useful analytics, generates automatic reports, and lets doctors update prescriptions directly. The goal is to improve communication, reduce mistakes, and keep patients safe.
